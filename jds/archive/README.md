# JDS Archive

This folder contains retired and superseded documents that are no longer in active use but are retained for reference and traceability.

## Rules

1. Documents are moved here when their status changes to **RETIRED**
2. Superseded documents may optionally be moved here (they can also stay in their original location with SUPERSEDED status)
3. **Never delete** archived documents — they are part of the historical record
4. The Document Registry maintains the definitive list of all document statuses
5. Archived documents retain their original JDS number permanently (numbers are never reused)

## Folder Structure

```
archive/
├── retired/          ← Documents no longer in use
├── superseded/       ← Documents replaced by newer versions
└── void/             ← Numbers assigned but documents never completed
```
