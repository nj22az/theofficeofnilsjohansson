## About This Game

Link's Awakening DX is the Game Boy Color enhanced version of The Legend of Zelda: Link's Awakening, originally released for the Game Boy in 1993. It was directed by Takashi Tezuka with a small team at Nintendo EAD. Notably, this was the first Zelda game set outside Hyrule and the first portable Zelda title.

The story takes place on the mysterious Koholint Island, where Link is stranded after a storm at sea. To escape, he must wake the Wind Fish by collecting eight Instruments of the Sirens from dungeons across the island. The game is remarkable for its introspective narrative and its surprising, bittersweet ending.

### Historical Significance

- **First portable Zelda** — proved the full Zelda experience could work on handheld hardware
- **First Zelda outside Hyrule** — the entire game takes place on Koholint Island
- **Started as an after-hours project** — development began as an unofficial side project by Nintendo staff experimenting with the Game Boy
- **Contains Nintendo cameos** — characters from Mario, Kirby, and other Nintendo series appear as friendly NPCs and enemies (Chain Chomps, Goombas, Kirby, etc.)
- **Influential game design** — the trading sequence side-quest and interconnected overworld influenced many later games
- **Critical acclaim** — widely considered one of the greatest Game Boy games ever made

### DX Enhancements (1998)

The DX version added:
- **Full colour** — the entire game was recoloured for the Game Boy Color palette
- **New dungeon** — the Colour Dungeon, accessible only on Game Boy Color, which awards a tunic upgrade (red for attack power, blue for defence)
- **Photo Mouse** — a side quest involving a photographer mouse who takes pictures at key moments. Photos could be printed using the Game Boy Printer accessory
- **Bug fixes** — several glitches from the original were fixed, including the screen warp exploit (partially)
- **Updated map screen** — colour-coded for easier navigation

### ROM Versions

Multiple ROM versions exist due to regional releases and bug fix revisions:

| Version | Region | Languages | Notes |
|---------|--------|-----------|-------|
| **v1.0** | Japan (JP) | Japanese | Initial Japanese release (DX version) |
| **v1.0** | USA | English | Initial North American release |
| **v1.0** | Europe | English, German, French | Initial European release (multilingual) |
| **Rev 1** | USA/Europe | English (+ DE/FR for EU) | Minor bug fixes |
| **Rev 2** | USA/Europe | English (+ DE/FR for EU) | Additional stability fixes |

**Note:** The original Game Boy version (1993) is a separate ROM entirely (.gb extension, non-colour). The DX version is backwards-compatible with the original Game Boy but runs in monochrome mode without the colour features.

### Known Differences Between Revisions

- **v1.0 → Rev 1:** Fixed several screen transition glitches. Minor text corrections.
- **Rev 1 → Rev 2:** Additional stability improvements. Fixed a rare save corruption issue.

The version can typically be identified by checking the ROM header bytes or verifying against No-Intro checksums.

## Game Structure

### Overworld

Koholint Island is divided into a grid of screens (16 x 16 map). Key locations:

| Location | Description |
|----------|-------------|
| **Mabe Village** | Starting village, home of Marin and Tarin |
| **Mysterious Forest** | Northwest forest, contains the Tail Cave key |
| **Goponga Swamp** | Swamp area, requires special item to traverse |
| **Tal Tal Heights** | Mountain range in the north |
| **Tal Tal Mountain Range** | Highest area, location of the Wind Fish's Egg |
| **Martha's Bay** | Southeast bay area with Catfish's Maw |
| **Animal Village** | Village of talking animals in the southeast |
| **Kanalet Castle** | Castle in the central-east area |
| **Rapids Ride** | River ride mini-game area |
| **Koholint Prairie** | Central grasslands |

## The Trading Sequence

One of the game's signature features — a chain of item trades spanning the entire island:

| # | Item | Given To | Location |
|---|------|----------|----------|
| 1 | Yoshi Doll | Mamasha (for Ribbon) | Mabe Village Trendy Game |
| 2 | Ribbon | Small BowWow (for Dog Food) | Mabe Village |
| 3 | Dog Food | Sale (for Bananas) | Sale's House O' Bananas |
| 4 | Bananas | Kiki the monkey (for Stick) | Kanalet Castle approach |
| 5 | Stick | Tarin (for Honeycomb) | Ukuku Prairie |
| 6 | Honeycomb | Chef Bear (for Pineapple) | Animal Village |
| 7 | Pineapple | Papahl (for Hibiscus) | Tal Tal Heights |
| 8 | Hibiscus | Christine (for Letter) | Animal Village |
| 9 | Letter | Mr. Write (for Broom) | Northwest of Mysterious Forest |
| 10 | Broom | Grandma Ulrira (for Fishing Hook) | Mabe Village |
| 11 | Fishing Hook | Fisherman (for Necklace) | Martha's Bay |
| 12 | Necklace | Mermaid (for Scale) | Martha's Bay |
| 13 | Scale | Mermaid Statue (for Magnifying Lens) | South Face Shrine area |
| 14 | Magnifying Lens | Enables reading the book in library | Mabe Village Library |

The Magnifying Lens is required to read the route through the Wind Fish's Egg — the final dungeon.

## Photo Mouse Locations (DX Exclusive)

The photographer mouse appears at various points to take photos. These could be printed with the Game Boy Printer. There are 12 photos total, triggered by specific actions or locations throughout the game.
