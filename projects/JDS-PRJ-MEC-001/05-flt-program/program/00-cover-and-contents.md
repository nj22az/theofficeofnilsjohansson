## How to Set Up a New Program

> You need: access to the site, vessel nameplates, and previous inspection certificates.

**Step 1** Fill in Doc 01 — write down who is responsible and sign it.

**Step 2** Fill in Doc 02 — walk the site, list every vessel, write down nameplate data, classify each one.

**Step 3** Fill in Doc 03 — for each vessel, think about what could go wrong and rate the risk.

**Step 4** Fill in Doc 04 — decide how often each check should be done, write down who will do it.

**Step 5** Print several copies of Doc 05 — you need one blank copy for every round.

> **Done.** You now have a compliant supervision program. Go do your first round with Doc 05.

---

## How to Run the Program

Every month (or quarter), take a blank Doc 05 to the site:

1. Check every vessel against the checklist
2. Write down readings and mark OK or X
3. If something is wrong, write it in the findings section
4. Sign and date the form, file it in this binder
5. At the end of the year, fill in Doc 07 to review how it went

## Regulatory Basis

This program satisfies **AFS 2017:3** (consolidated with AFS 2019:1, 2020:10, 2022:2). The compliance reference document at the back of this binder maps every section to the regulation.

> **Warning:** Non-compliance penalty: **10,000 — 100,000 SEK.** Operating without documented supervision routines is a sanctionable offence under 4 Kap. §17.
